# AI-First CRM HCP Interaction Management Module

## Tech Stack

- React
- FastAPI
- LangGraph
- Groq LLM
- SQLite

## Features

- Log Interaction
- Edit Interaction
- Search HCP
- Generate Summary
- Schedule Follow-up

## How to Run

### Backend

cd Backend
pip install -r requirements.txt
uvicorn app.main:app --reload

### Frontend

cd frontend
npm install
npm run dev
